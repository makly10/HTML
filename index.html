ETIQUETAS DE FORMATO

Este tipo de etiquetas son aquellas que permiten establecer elementos dentro de la página y así poder dar jerarquías de títulos, parrafos, listas, etc.

Ejemplo de etiquetas de formato:

  1. etiqueta de párrafos (p)
  2. etiquetas de lista ordenada (ol)
  3. etiquetas de lista sin orden (ul)

ETIQUETAS DE BLOQUES

Este tipo de etiqueta se utiliza para crear "secciones" de contenido, separación vertical en el diseño.

Ejemplos de etiquetas de bloques:

  • etiqueta header (encabezado)
  • etiqueta main (contenido principal)
  • etiqueta footer (pie de página)

ETIQUETAS DE CONTENIDO

Este tipo de etiqueta se utiliza para definir tipos de elementos (texto, imágen, listas, tablas, etc) para que puedan ser interpretadas por el navegador.

Además las etiquetas tienen "atributos" que permiten controlar al elemento propiamente dicho.

Ejemplos de etiquetas de contenido:

  • etiqueta "img": permite agregar una imagen y su atributo es "src".
  • etiqueta "a": permite enlazar a otra página o archivos, su atributo es "href".
  • etiqueta "input": permite que el usuario pueda ingresar información. Sus atributos son "type" y "name".
  • etiqueta "label": acompaña a la etiqueta input para indicar lo que quiero que ingrese el usuario, su atributo es "for".
